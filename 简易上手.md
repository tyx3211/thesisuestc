1. 大家先clone本项目仓库

    ```bash
    git clone https://github.com/tyx3211/thesisuestc.git
    cd thesisuestc
    ```

2. 大家在ubuntu上全量安装texlive

    ```bash
    sudo apt update
    sudo apt install texlive-full
    ```

3. 然后在项目根目录随便创建你的一个tex文档（下文就叫report.tex），然后让AI按照Readme.md中的模板用法说明以及main.tex中的用法实例，就能帮你写一份符合格式要求的文档了。
    * 注意：参考文献让AI照着main.tex中引用的reference.bib的格式写（可新创一个文件叫ref.bib），然后让AI用main.tex中的方式引入ref.bib。

4. 然后在项目根目录编译运行（用latexmk指定xelatex）：

    ```bash
    latexmk -xelatex report.tex
    ```
